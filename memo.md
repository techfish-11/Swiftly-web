# カード
- 高性能サーバー
    オンプレミスで運用される高品質なインフラストラクチャ
- 透明性の高い運用
    オープンソースで、デプロイまでGitHubでオープン。
- エンゲージメント向上
    サーバーを活性化させる楽しいインタラクション機能
- AI分析機能
    高度なAIによるサーバー分析と未来予測
- 開発者ツール
    プログラマーのための便利なユーティリティ機能
- 音声機能
    高性能な音声読み上げシステム
- 匿名投票機能
    Discord標準ではできない匿名投票機能を実装。
- 荒らし対策
    短縮URLにも対応した招待リンクのブロック機能に加え、新規作成アカウントや初期アイコンのメッセージを自動削除。設定のオン/オフも可能。
- 時報
    １チャンネル３つまで設定可能の時報
